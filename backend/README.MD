# Medicine Tracker (MERN)

Personal medicine reminder & tracker app for CDAC PG-DAC placements.

## Current Status (as of Jan 18, 2026)
- Backend: Express + MongoDB Atlas
- User auth: Register/Login with JWT + bcrypt
- Protected CRUD: POST & GET /api/meds (linked to authenticated user)
- Next: PUT/DELETE meds, expiry warnings, intake logging, React frontend

## Run locally
cd backend
npm install
npm start
